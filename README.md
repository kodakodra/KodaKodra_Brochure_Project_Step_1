# KodaKodra — Step 1: Single Page HTML Landing

## What This Is
A single page portfolio landing site for KodaKodra, a freelance web development brand. Built as the foundation for an iterative portfolio progression — every subsequent step builds on top of this base.

## What This Demonstrates
- Semantic HTML5 structure
- Bootstrap 5 grid and responsive layout
- Font Awesome icons
- Custom CSS via separate stylesheet
- Mobile-first design with collapsible navbar
- Consistent branding and visual identity across sections
- Clean, readable code with no unnecessary dependencies

## Pages
- `index.html` — single page with anchored sections: Hero, About, Services, Stack, Contact, Footer

## Tech Used
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome 6

## Notes
- No JavaScript, no PHP, no database — intentionally static at this stage
- Contact section is informational only — no form yet
- All subsequent steps build on this same base without retrofitting
